# Matias Di Girolamo | Portfolio

Portfolio personal de Matias Di Girolamo, desarrollado para presentar mi perfil como desarrollador FullStack .NET, mi experiencia profesional, mi stack principal y una seleccion de proyectos reales.

El sitio esta pensado como una carta de presentacion digital: una forma simple y directa de mostrar quien soy, en que tecnologias trabajo y como me muevo entre backend, frontend y producto segun lo que necesite cada proyecto.

## Sobre Mi

Soy Matias Di Girolamo, desarrollador FullStack de Buenos Aires, Argentina. Trabajo principalmente con .NET, C#, ASP.NET, SQL Server, APIs REST, MVC y tambien con tecnologias como React, Node.js, Prisma, Tailwind y Docker en proyectos personales y productos web completos.

Actualmente trabajo en IPN ERP Cloud, donde participo en desarrollo de software empresarial y soluciones orientadas a procesos reales de negocio.

## Sobre El Portfolio

Este portfolio fue disenado y desarrollado para reflejar mi perfil de una forma mas personal y profesional, evitando el estilo generico de template.

Incluye:

- Presentacion bilingue en espanol e ingles
- Tema claro y oscuro
- Secciones de experiencia, stack y proyectos
